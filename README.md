# IsaiGarciaPrueba

Proyecto base con:

- `frontend/`: Vue 3 + Vite + Bootstrap 5
- `backend/`: ASP.NET Core Web API (.NET 8)
- `.env`: configuracion de SQL Server local

## Ejecutar backend

```powershell
cd backend
dotnet run
```

## Ejecutar frontend

```powershell
cd frontend
npm run dev
```

## Endpoints

- `GET /api/health`
- `GET /api/database`
